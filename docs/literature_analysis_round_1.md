## 论文核心方法摘要

### 1. ChemCrow (Bran et al., 2024)
- **技术要点**: 集成18个专家化学工具的 Toolbox-based Agent
- **关键洞察**: 工具丰富度直接决定 Agent 能力边界
- **本项目映射**: 当前仅有4个化学工具（生成、对接、逆合成、评估），工具链单薄

### 2. ChemAgents (Song et al., 2025)
- **技术要点**: 分层多Agent系统（Manager + 5个 Specialist）
- **关键洞察**: Hierarchical delegation 可管理大规模工具集，避免单Agent过载
- **本项目映射**: 当前是单脚本 pipeline，无分层协作

### 3. LARC (Baker et al., 2025)
- **技术要点**: Agent-as-a-Judge 逆合成框架
- **关键洞察**: 规则覆盖率是逆合成质量的关键决定因素；路线需要被评审可行性
- **本项目映射**: synthesis_v2.py 已有35条规则，但缺乏多步递归和路线评审

### 4. MOOSE-Chem (Yang et al., 2025)
- **技术要点**: 进化算法导航假设空间，LLM作为变异算子
- **关键洞察**: 闭环进化搜索是探索化学空间的高效方式
- **本项目映射**: pipeline.py 已有2代进化，但每代内部仍是盲生成，缺乏适应度引导

### 5. Coscientist (Boiko et al., 2023)
- **技术要点**: 多模块Planner（Web搜索+文档检索+代码执行+实验执行）
- **关键洞察**: 迭代反思和错误自纠正能力是自主科研的核心
- **本项目映射**: 运行完即结束，无自动分析-调整闭环
